get_basic_type

Description:

public DeviceType get_basic_type ()

Gets the basic type of a device type.

Applies the type mask on a DeviceType to get its basic type. If the device type is already basic, it will return the same type.

E.g DeviceType.JOYPAD_MULTITAP.get_basic_type () returns DeviceType.JOYPAD, and DeviceType.JOYPAD.get_basic_type () also returns Type.JOYPAD.

Returns:

the basic type of a device type
