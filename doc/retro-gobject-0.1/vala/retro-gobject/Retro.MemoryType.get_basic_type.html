get_basic_type

Description:

public MemoryType get_basic_type ()

Gets the basic type of a memory type.

Applies the type mask on a MemoryType to get its basic type. If the memory type is already basic, it will return the same type.

E.g MemoryType.SNES_BSX_RAM.get_basic_type () returns MemoryType.SAVE_RAM, and MemoryType.SAVE_RAM.get_basic_type () also returns MemoryType.SAVE_RAM.

Returns:

the basic type of a memory type
