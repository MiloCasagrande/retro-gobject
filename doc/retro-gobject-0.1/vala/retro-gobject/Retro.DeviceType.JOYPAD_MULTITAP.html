JOYPAD_MULTITAP

Description:

JOYPAD_MULTITAP = 1 << 8 | JOYPAD

A joypad multitap.

It is a specialization of the DeviceType.JOYPAD.

It should only be used in Core.set_controller_port_device to inform Libretro implementations about use of a joypad multitap.

In input state callback, however, the device will be DeviceType.JOYPAD.
