audio_request

Description:

public void audio_request () throws CbError

Notifies the core that it can play audio.

Used by core where audio can be asynchronous to let the frontend ask for audio to play. The audio data will then be sent via the usual methods.

Exceptions:

CbError

the core or its callback couldn't be found
