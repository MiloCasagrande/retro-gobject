replace_image_index

Description:

public bool replace_image_index (uint index, GameInfo info)

Replaces the disk image associated with index.

Virtual disk tray must be ejected when calling this.

Arguments to pass in info have same requirements as Core.load_game.

Parameters:

index

index of the disk image to replace

info

information on the disk image to use

Returns:

true on successfully replaced image, false otherwise
