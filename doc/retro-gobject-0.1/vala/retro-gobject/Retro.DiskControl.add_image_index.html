add_image_index

Description:

public bool add_image_index () throws CbError

Adds a new valid index to the internal disk lit.

This will increment subsequent return values from get_num_images by 1. This image index cannot be used until a disk image has been set with replace_image_index.

Returns:

true on successfully added index, false otherwise

Exceptions:

CbError

the core or its callback couldn't be found
